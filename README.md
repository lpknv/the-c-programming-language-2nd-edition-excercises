# The C Programming Language - 1st Edition - Exercises

This repository contains some exercises of the book The C Programming Language.
Feel free to ramp up your C programming understanding.

## Usage
Here is an example on how to create (or run if already exists) a new folder for an exercise or chapter.

### Linux
```shell
build.sh ch-2.1 "Title for a chapter"
```

### Windows
```shell
build.bat ch-2.1 "Title for a chapter"
```
